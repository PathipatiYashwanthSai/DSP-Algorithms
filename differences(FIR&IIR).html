1.FIR filter stands for finite impulse response filter. 1.IIR filter stands for infinite impulse response filter.
2.Design is based on all the infinite samples of the impulse response. 2.Design is based on only a finite number of samples of impulse response.
3.The impulse response cannot be directly converted to digital filter transfer function. 3.The impulse response can be directly converted to digital filter transfer function.
4.The digital filter can be directly designed. First analog filter is to be designed and it has to be transformed to a digital filter. 4.The digital filter can be directly designed.
5.The specifications include the desired characteristics for magnitude response only. 5.The specifications include the desired characteristics for both magnitude and phase response.
6.Linear phase characteristics cannot be achieved . 6.Linear phase charactyeristics can be achieved.
